# Hylight — サビだけ、次々と。

検索せずに「良い曲」に出会うための音楽ディスカバリーサイト。
YouTube 公式チャンネルの楽曲を **サビから** 再生し、**5〜20秒**で次の曲へ自動で切り替わります。

## ターゲット
- 自分から検索する気はないが、良い曲を早く見つけたい人（検索疲れ）
- 自分にマッチする音楽を探している人

## 特徴
- **サビ開始**：どの曲もサビ（インストは盛り上がり/フック）から再生。
- **再生秒数**：5 / 10 / 15 / 20 秒（デフォルト 5秒）。時間が来ると自動で次へ。
- **スムーズ切替**：2つのプレイヤーを重ねて裏でプリロードし、クロスフェードで遷移。
- **フィルタ**：ジャンル / 雰囲気 / 歌あり・インスト / 言語（邦楽・洋楽）。
- **おすすめアルゴリズム**：特徴ベクトル（ジャンル/雰囲気/言語/ボーカル/アーティスト）の
  選好スコアで重み付け選曲。
  - 👍 いいね・動画クリック ＝ **良い** → 似た曲を優先
  - 👎 最後まで再生（自動切替）・途中スキップ ＝ **好みでない** → 違う曲
  - ↩ 前の動画に戻ってクリックすると **良い** に変更
- **動画クリック**で公式 YouTube が新規タブで開きます。

## 使い方（ローカル）
ブラウザの自動再生・iframe の制約があるため、簡易サーバ経由での起動を推奨します。

```bash
# どれか一つ
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

開始ボタンを押すと再生が始まります（音声付き自動再生のため最初の操作が必要）。

## 楽曲の追加・編集
`data.js` の `window.SONGS` 配列を編集してください。

| フィールド | 説明 |
|---|---|
| `id` | YouTube videoId（`https://youtu.be/<id>`）。公式チャンネルのものを使用。 |
| `title` / `artist` | 表示名 |
| `genre` | `pop` / `rock` / `rnb` / `dance` / `ballad` |
| `mood` | `bright` / `dark` / `sad` / `chill` / `energetic` |
| `lang` | `jp`（邦楽）/ `en`（洋楽） |
| `vocal` | `vocal`（歌あり）/ `inst`（インスト） |
| `chorusStart` | サビ開始秒。ここから再生します。 |

> 収録曲は公式チャンネルのアップロードを想定しています。再生できない曲があれば
> その行を差し替え・削除してください。
